# ⛓️ BlockDrive — Decentralized P2P Secure Storage

Upload files that are **encrypted**, **sharded**, and **distributed** across a peer-to-peer network with a blockchain ledger.

## Features
- 🔐 AES-256 client-side encryption (Fernet)
- ✂️ File sharding into 1 MB chunks
- 🌐 Distributed storage across P2P nodes
- ⛓️ Blockchain ledger for file tracking
- 🌳 Merkle tree integrity verification
